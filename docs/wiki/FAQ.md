# FAQ

## Is it scrapers or scrappers?

`Scrapers` is correct.

## Do I need OpenAI to use this repo?

No. AI enrichment is optional and only affects the file-based webpage workflow.

## Can I export CSV?

Yes. Use `--save-format csv` or `export --format csv`.

## Can I run this in Docker?

Yes. The repository now includes a `Dockerfile`.
