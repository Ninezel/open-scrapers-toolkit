# Roadmap

This repository is intended to become a broad open-source scraper collection rather than a one-off code dump.

## Next expansion areas

- more official government feeds
- health-agency report sources
- NGO and humanitarian situation-report feeds
- additional academic indexes
- CSV and NDJSON exporters
- Docker packaging
- scheduled jobs and GitHub Actions examples
- lightweight parser tests
- source health checks

## Suggested next scrapers

- WHO news or publications feeds
- UN News topic feeds
- OECD reports
- government open-data portals
- policy paper indexes
- journal TOC feeds
- environment and climate data APIs

## Contribution goal

Grow beyond 25 scrapers without turning the repo into a tangle of one-off scripts. Shared helpers, docs, and source rules are what make that possible.
